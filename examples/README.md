# Example Configurations

These are larger, more comprehensive default configs for users who want broader mod compatibility out of the box. The mod's built-in defaults are minimal — just vanilla and a few common keyword patterns. These examples extend that to cover ~25 common mod-added materials.

## How to use

Copy the contents of one of these files into your `config/phats_progression/block_tiers.json` or `config/phats_progression/tool_tiers.json`, replacing the auto-generated default. Or merge entries into your existing config.

These configs assume the standard tier progression: 1=wood, 2=stone, 3=copper, 4=iron, 5=diamond, 6=netherite, with materials assigned based on common mod conventions and rarity. Tier assignments may not match every mod's intent — modpack devs should adjust as needed.

## Files

- `expansive_tool_tiers.json` — Comprehensive tool/weapon tier assignments
- `expansive_block_tiers.json` — Comprehensive ore block tier assignments

## Reporting issues

If a material is in the wrong tier or missing entirely, open an issue or PR.
